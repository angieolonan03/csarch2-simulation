Extended Booth's Algorithm

The extended Booth's algorithm can reduce the number of intermediate summands

by half when multiplying two signed binary numbers in two's complement notation.

Here are the steps to perform extended boothâ€™s algorithm:

A. Convert the operands to two's complement:

1. Use the minimum number of bits required to represent a decimal input.

2. Use the number of bits specified for a binary input.

B. Sign-extend the operands if necessary to ensure they have the same number of bits.

C. Convert the multiplier to its extended Booth's equivalent:

1. Append a 0 to the least significant bit.

2. If the number of bits in the multiplier is odd, perform sign extension.

3. Use bit-pair recoding starting from the least significant bit.

D. Multiply using the pencil-and-paper method, ignoring the additional step even if

the multiplier is negative. Since each bit-pair is equivalent to two bits,

skip two bits for each intermediate product
